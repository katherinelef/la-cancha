## La cancha

Para realizar el ejercicio se utilizo la propiedad ***Positioning***.

1. Se creo una clase ***.general*** para el fondo de la cancha:        
  ***position: relative***

2. Se creo una clase ***.subgeneral*** para el marco de la cancha:
  ***position: absolute***

3. Se crearon dos arcos y el circulo central:
  ***position:relative***

4. Se crearon tres pelotas:
  ***position: absolute***
